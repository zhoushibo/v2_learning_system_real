## 🚀 日常使用流程

### 早晨启动
```
1. 双击运行：python main_v3.py
2. 切换到 [🚀 快速启动]
3. 点击 [🚀 一键启动服务]
4. 等待 10 秒完成
5. ✅ 开始工作！
```

### 工作中
- **查知识：** [📚 知识库] → 输入关键词 → 搜索
- **学新知：** [🧠 V2 学习] → 输入主题 → 开始学习
- **看状态：** [📊 仪表盘] → 查看服务状态
- **管项目：** [📁 项目管理] → 更新进度

### 晚上离开
- **短暂离开：** 最小化到托盘（后台运行）
- **长时间：** 右键托盘 → [⏹️ 停止所有服务]
- **下班：** 右键托盘 → [❌ 退出]

## 🔧 常见问题速查

### ❌ 服务启动失败
```
解决：[🔍 智能诊断] → [🔍 开始诊断] → [🔧 一键修复]
```

### ❌ 端口被占用
```
解决：[🔍 智能诊断] → 查看占用进程 → [🔧 停止占用进程]
```

### ❌ 知识库搜索无结果
```
解决：
1. 检查服务是否运行（[📊 仪表盘]）
2. 导入文件到知识库
3. 换个关键词再试
```

### ❌ V2 学习失败
```
检查：
1. v2_learning_system_real 目录存在
2. API Key 已配置
3. 网络连接正常
```

## 💡 使用技巧

### 高效技巧
1. **双击托盘图标** - 快速恢复窗口
2. **搜索时按回车** - 不用点搜索按钮
3. **学习历史自动保存** - 随时查看之前学过的内容
4. **配置编辑自动备份** - 改错了可以恢复

### 省时技巧
1. **一键启动** - 不用手动启动每个服务
2. **智能诊断** - 自动发现问题并修复
3. **系统托盘** - 最小化到后台，不中断服务
4. **主题切换** - 根据环境快速调整

### 进阶技巧
1. **V2 学习模式选择：**
   - 快速学习（10 秒）- 快速了解
   - 深度学习（30 秒）- 日常使用 ✅
   - 全面学习（60 秒）- 深入研究

2. **知识库搜索技巧：**
   - 使用具体关键词
   - 支持中英文混合
   - 支持技术术语

3. **项目管理技巧：**
   - 定期更新完成度
   - 记录缺失项和下一步
   - 按状态排序查看
