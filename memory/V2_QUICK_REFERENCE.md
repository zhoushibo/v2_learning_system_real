## ⚡ 快速判断（10秒内）

```
必须使用V2，如果：
✅ 任务时间 > 1分钟
✅ 需要并发处理
✅ 需要流式输出
✅ 会卡住/等待
✅ 需要后台执行

可选使用V2，如果：
⚪ 任务时间 < 10秒
⚪ 单个任务
⚪ 偶尔使用
```
