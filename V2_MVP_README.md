**统一架构：** Gateway 统一管理 6 个 AI Provider --- ## 🏗️ 系统架构 ``` V2 MVP 全能 AI 系统 ├── 🧠 V2 学习系统 (v2_learning_system_real/) │ ├── LearningEngine - 3 Worker 并发学习 │ ├── 自动去重更新 - 基于主题 + 视角 │ └── 知识库集成 - 学习完成自动保存 │ ├── 📚 知识库系统 (knowledge_base/) │ ├── ChromaDB - 语义搜索 │ ├── FTS5 - 关键词搜索 │ ├── Web UI - http://localhost:8501 │ └── CLI - 命令行工具 │ ├── 🌐 Gateway 架构 (openclaw_async_architecture/) │ ├── 统一 Gateway 服务 (端口 8001) │ ├── 6 个 AI Provider (zhipu, hunyuan, nvidia1-3, siliconflow) │ └── 自动故障切换 │ └── 🛠️ 工具引擎 │ ├── web_search - 网络搜索 │ ├── web_fetch - 网页抓取 │ └── exec - Shell 命令执行 ``` --- ## ✨ 核心功能 ### 1. **V2 学习系统** **特点：** - 3 Worker 并发学习，效率提升 3 倍 - 多视角学习（technical, practical, theoretical, historical, comparative） - 学习完成自动保存到知识库 - 自动去重更新（基于主题 + 视角） **使用示例：** ```python from v2_learning_system_real import LearningEngine import asyncio async def main(): engine = LearningEngine(num_workers=3) # 学习并自动保存 results = await engine.parallel_learning( "Python 异步编程", num_perspectives=3, save_to_kb=True ) print(f"✅ 学习了 {len(results)} 个视角") asyncio.run(main()) ``` ### 2. **知识库系统** **特点：** - **双索引：** ChromaDB（语义）+ FTS5（关键词） - **Web UI：** Streamlit 界面，http://localhost:8501 - **CLI：** 命令行导入和搜索 - **自动去重：** 基于主题 + 视角的智能去重 **使用示例：** ```bash # 搜索知识 python knowledge_base/cli.py search "Python 异步" # 导入文件 python knowledge_base/cli.py import document.md ``` ### 3. **Gateway 架构** **特点：** - **统一入口：** ws://127.0.0.1:8001 - **6 个 Provider：** - zhipu (glm-4-flash) - 最快，1.03s - hunyuan (hunyuan-lite) - 256K 上下文 - nvidia1-3 (glm4.7) - 深度思考 - siliconflow (bge-large-zh) - Embeddings 专用 **配置：** `openclaw_async_architecture/API_CONFIG_FINAL.json` --- ## 📊 性能指标 | 指标 | 数值 | 说明 | |------|------|------| | **学习速度** | ~21s | 3 视角学习 + 保存 | | **效率提升** | 178× | 相比手动学习 | | **测试覆盖率** | 96% | 单元测试 | | **Gateway 延迟** | 661ms | 平均响应时间 | | **Embeddings** | 0.1s | SiliconFlow API | | **并发 Worker** | 3 | 同时学习 | --- ## 🚀 快速开始 ### 1. 运行整合演示 ```bash python v2_mvp_demo.py ``` **输出：** -
